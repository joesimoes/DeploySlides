Web Servers and App Servers

What is a Web Server?

  • App Built on Layers
  • First level is the Web Server
  • HTTP Protocol
  • Static Requests
  • Load Balancer

    EC2 (Elastic Compute Cloud)

  • Part of Amazon's CCP
  • Allows Users to Rent Virtual Computers to Run Apps
  • Elastic Load Balancer

Load Balancer

  • Distributes Incoming Traffic
  • Maximizes Output, Minimizes Response Time

What is an App Server?

  • Provides Business Logic
  • Scalability and Fault Tolerance

Nginx

  • High Concurrency
  • Asynchronous
  • Load Balancing
  • Non-Linear Scalability

Unicorn

Unicorn

  • Application Server Written in Ruby
  • Hosts Rack Apps Generating Dynamic Content

    Rack

  • Middleware Which Sits Between App and HTTP Requests

Capistrano

  • Automates Process of Making App Available on Web Server
  • Execution of Commands on Remote Machines via SSH
  • Uses DSL Borrowed from Rake
  • Recipes to Deploy Changes or Roll Back Changes

SSH

  • Cryptographic Network Protocol - Secure Data
  • Access to Shell Accounts on Unix OS
  • Public-Private Key Pair Encryption

Takeaways

  • Some of the Most Efficient Technologies Available
  • Enable Growth of Some of the Largest Sites
